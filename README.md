# projet_calcul_temps_gpx

## Utilisation

Charger un GPX

### L'appli vous donne :
- Profil de la course
- Découpage intelligent des sections selon le terrain (possibilité d'ajuster selon le besoin avec les indicateurs latéraux)

### Algorithme côte :
- Vous donnez une côte UTMB visée : estimation du temps à réaliser pour atteindre cette côte
- Calcul automatique du temps (et allure) à respecter sur les sections
